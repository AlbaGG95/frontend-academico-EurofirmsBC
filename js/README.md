# JavaScript

Esta carpeta agrupa los scripts del proyecto.

## Archivos actuales

- `theme-toggle.js`: controla el cambio de tema
- `js-profile-message.js`: ejercicio del mensaje personal
- `js-variable-types.js`: ejercicio de variables y tipos
- `js-multiples-of-7.js`: ejercicio de bucles y conteo
- `js-sumar-hasta.js`: ejercicio de funciones con suma acumulada

## Regla de uso

- si un ejercicio necesita logica propia, crea aqui su archivo `.js`
- usa nombres que indiquen claramente el ejercicio o la funcionalidad
- evita mezclar logica comun con logica de ejercicios concretos
