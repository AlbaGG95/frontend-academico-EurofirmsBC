# Assets

Esta carpeta contiene recursos estaticos usados por el proyecto.

## Subcarpetas

- `gallery/`: imagenes para ejercicios de galeria
- `profile/`: avatar e imagenes de perfil

## Regla de uso

- guarda aqui imagenes, iconos o recursos visuales reutilizables
- manten los nombres claros para que sea facil localizarlos
- si se crea un nuevo bloque de recursos, anade una subcarpeta especifica
