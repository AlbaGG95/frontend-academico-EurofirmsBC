# CSS

Esta carpeta centraliza las hojas de estilo compartidas del proyecto.

## Archivos

- `hub-pages.css`: estilos comunes para hubs, portada y documentacion
- `exercise-theme.css`: estilos comunes para ejercicios individuales

## Regla de uso

- si un estilo sirve para varias paginas del proyecto, debe vivir aqui
- evita crear estilos duplicados dentro de cada HTML si pueden compartirse
- los estilos puntuales de una pagina pueden mantenerse inline solo si son
  pequenos y muy especificos de esa vista
