Arrays y objetos

Aquí dejo la base de estructuras de datos que más voy a usar.

Esta ficha me sirve para ordenar mejor la información y trabajarla con métodos sin perderme en datos sueltos.

Ficha de estudio

Lo que quiero tener claro

Objetivo

Guardar datos con estructura

Quiero dejar de trabajar solo con valores sueltos y empezar a pensar en colecciones y propiedades.

Error comun

Confundir array con objeto

Un array me sirve para listas ordenadas; un objeto, para agrupar propiedades con nombre.

Regla

Elegir la estructura antes del metodo

Antes de pensar en `map`, `filter` o acceso por punto, necesito saber si mis datos estan bien modelados.

Base

Conceptos y métodos que quiero dominar

Array

Listas ordenadas de elementos. Metodos útiles: `push`, `map`, `filter`, `find`.

Objeto

Conjunto de propiedades con clave y valor. Acceso comun: `objeto.propiedad`.

Combinacion

Muy frecuente: arrays de objetos para representar listas de tareas, productos o usuarios.

Ejemplo minimo

Array de objetos

const tareas = [ { id: 1, titulo: "Repasar flexbox", hecha: true }, { id: 2, titulo: "Practicar DOM", hecha: false } ]; const pendientes = tareas.filter((tarea) => !tarea.hecha);
