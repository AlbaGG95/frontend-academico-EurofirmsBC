Lo que quiero revisar antes de bloquearme

Esta página funciona mejor si la leo como una lista de comprobacion rápida, no como teoría larga.

Rutas
  • Comprobar que el archivo existe de verdad.
  • Usar rutas locales claras como ./archivo.html.
  • No depender de `127.0.0.1` si solo estoy abriendo HTML estatico.
HTML
  • Revisar etiquetas sin cerrar.
  • Comprobar anidacion.
  • No abusar de `div` cuando una etiqueta semántica encaja mejor.
CSS
  • Poner `box-sizing: border-box`.
  • Probar `gap` antes de pelearme con margenes.
  • Revisar el padre flex antes de tocar los hijos.
