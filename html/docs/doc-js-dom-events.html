DOM y eventos

Aquí junto la parte de interacción con la página.

Esta ficha me interesa porque conecta la lógica con el navegador y me prepara para formularios, botones y cambios visuales.

Ficha de estudio

Lo que quiero tener claro

Objetivo

Selecciónar, escuchar y cambiar

Quiero pensar el DOM como una secuencia simple: seleccióno un nodo, escucho un evento y cambio algo.

Error comun

No selecciónar bien el elemento

Si el selector falla o devuelve `null`, todo lo demas se rompe aunque la lógica sea correcta.

Regla

Probar primero con un boton simple

Antes de hacer algo complejo, me ayuda validar el flujo con un click y un cambio pequeno.

Base

Conceptos que quiero dominar

Selección

`querySelector` y `querySelectorAll` para localizar elementos.

Eventos

`addEventListener` para reaccionar a clicks, input, submit y otros eventos.

Cambios

`textContent`, `innerHTML`, clases o estilos para actualizar la interfaz.

Ejemplo minimo

Click que cambia texto

const boton = document.querySelector(".boton"); const titulo = document.querySelector(".titulo"); boton.addEventListener("click", () => { titulo.textContent = "Texto actualizado"; });
