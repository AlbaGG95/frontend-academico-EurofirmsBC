CSS

Aquí organizo la base teórica de CSS por bloques.

En vez de meter todo en una sola página, prefiero usar esta ficha como mapa general y separar cada tema grande en su propia ruta de estudio.

Objetivo
Orden

Si separo bien los temas, luego encuentro más rápido una duda de selectores, de caja o de posicionamiento.

Mapa de estudio

Cómo quiero dividir CSS

Esta estructura me resulta más académica porque cada tema tiene su propio espacio y no se mezcla con el resto.

Tema 01

Selectores CSS

Clases, ids, descendientes, hijos directos y pseudoclases. Es la base para escribir CSS con sentido.

Abrir selectores CSS Estructura de reglas y ejemplos
Tema 02

Box model y position

Margenes, padding, border, width, height y formas de posicionar elementos dentro del flujo.

Abrir box model y position Caja, flujo y posiciones
Tema 03

Flexbox

Lo dejo separado porque lo estoy usando mucho en prácticas reales y quiero verlo también con ejemplos visuales.

Abrir Flexbox Propiedades, criterios y demos
Criterio

Cómo quiero estudiar CSS aquí

Paso 1

Entender el concepto

Antes de memorizar propiedades, quiero saber para que sirve cada bloque.

Paso 2

Ver un ejemplo minimo

Si un concepto no tiene ejemplo, me cuesta más recordarlo luego en una tarea.

Paso 3

Anotar errores comunes

Si se exactamente donde suelo fallar, aprendo más rápido y repito menos errores.
