CSS

Aqui organizo la base teorica de CSS por bloques.

En vez de meter todo en una sola pagina, prefiero usar esta ficha como mapa general y separar cada tema grande en su propia ruta de estudio.

Objetivo
Orden

Si separo bien los temas, luego encuentro mas rapido una duda de selectores, de caja o de posicionamiento.

Mapa de estudio

Como quiero dividir CSS

Esta estructura me resulta mas academica porque cada tema tiene su propio espacio y no se mezcla con el resto.

Tema 01

Selectores CSS

Clases, ids, descendientes, hijos directos y pseudoclases. Es la base para escribir CSS con sentido.

Tema 02

Box model y position

Margenes, padding, border, width, height y formas de posicionar elementos dentro del flujo.

Flexbox

Lo dejo separado porque lo estoy usando mucho en practicas reales y quiero verlo tambien con ejemplos visuales.

Abrir Flexbox Propiedades, criterios y demos
Criterio

Como quiero estudiar CSS aqui

Paso 1

Entender el concepto

Antes de memorizar propiedades, quiero saber para que sirve cada bloque.

Paso 2

Ver un ejemplo minimo

Si un concepto no tiene ejemplo, me cuesta mas recordarlo luego en una tarea.

Paso 3

Anotar errores comunes

Si se exactamente donde suelo fallar, aprendo mas rapido y repito menos errores.
