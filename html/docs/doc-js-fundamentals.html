Ficha de estudio

Lo que quiero tener claro

Objetivo

Entender la lógica antes que memorizar

Quiero saber que estoy intentando resolver antes de pensar en la sintaxis exacta.

Error comun

Escribir sin pensar el flujo

Si no tengo claro el orden de pasos, las condiciones o los datos, el código se enreda rápido.

Regla

Primero datos, luego acciones

Me ayuda separar que información tengo, que quiero hacer con ella y que resultado espero obtener.

Base

Conceptos que quiero dominar

Variables

`let` para valores que cambian, `const` para valores que no quiero reasignar.

Condicionales

`if`, `else if` y `else` para decidir entre caminos.

Bucles

`for` y `while` para repetir acciones con control.

Funciones

Me sirven para encapsular una accion y reutilizarla.

Funcion simple con condicion
