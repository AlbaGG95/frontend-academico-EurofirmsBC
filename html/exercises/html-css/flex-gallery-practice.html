Práctica continúa

Galería de fotos con estructura de bloques y continuidad visual.

Esta práctica representa una segunda fase del ejercicio de bloques. Mantiene la lógica de Flexbox y la amplía con imágenes reales, control del recorte visual y comportamiento responsive.

Aplicación de la fase anterior

Cada bloque pasa a ser un cuadro fijo de 200x200 y la galería conserva el reparto del espacio mediante Flexbox. Esta continuidad evita rehacer el ejercicio desde cero y muestra una evolución natural del mismo encargo.

Estructura

Cuadros de medida fija

Cada imagen se inserta dentro de un contenedor estable para facilitar el reparto y la comparación entre filas.

Imagen

Recorte sin deformación

object-fit: cover llena el cuadro manteniendo una presentación limpia y homogénea.

Continuidad

Misma tarea, nueva complejidad

La galería demuestra cómo una práctica previa puede escalar hacia un resultado más completo sin perder coherencia documental.
