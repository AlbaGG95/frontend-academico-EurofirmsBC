Ejercicio base de Flexbox

Distribución de bloques con wrap y separación horizontal.

Esta práctica funciona como ejercicio fundacional del módulo. Sirve para entender cómo se reparte el espacio en un contenedor flex, cómo actúa el salto de línea y por qué esta base es útil para tareas más complejas.

Resultado visual del layout

Los seis bloques muestran una distribución equilibrada con display: flex, justify-content: space-between y flex-wrap. Es una base que luego puede evolucionar hacia galerías, cards o paneles de contenido.

Objetivo

Comprender el contenedor flex

La práctica enseña a pensar primero en el contenedor y después en la distribución de los elementos hijos.

Técnica

Reparto con space-between

justify-content: space-between reparte el espacio libre entre los elementos de cada fila.

Continuidad

Base para otras fases

Este mismo ejercicio puede ampliarse con imágenes, cards o módulos de contenido sin perder la lógica inicial.
