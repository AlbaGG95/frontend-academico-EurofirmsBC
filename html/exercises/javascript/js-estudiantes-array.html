Arrays + objetos

Gestion basica de estudiantes con un array de objetos.

Esta practica crea un array llamado estudiantes con varios objetos y una funcion agregarEstudiante(nombre, edad, notas) que inserta un nuevo estudiante en la coleccion.

Que se practica

El objetivo es acostumbrarse a trabajar con arrays de objetos, propiedades accesibles por nombre y una funcion que modifica la estructura de datos sin perder claridad.

Array

Coleccion inicial

El ejercicio parte de varios estudiantes ya guardados con nombre, edad y notas.

Funcion

Agregar un nuevo objeto

La funcion recibe tres datos y construye un nuevo objeto para insertarlo con push dentro del array.

Resultado

Array actualizado

Despues de ejecutar la funcion, el nuevo estudiante queda disponible junto al resto de elementos del array.

Vista tipo consola

Este bloque muestra el array inicial, el nuevo estudiante agregado y el estado final de la coleccion.
