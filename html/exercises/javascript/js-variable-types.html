Primer ejercicio de JavaScript

Variables, reasignaciones y tipo de dato en JavaScript.

Esta práctica trabaja una sola variable para comprobar cómo cambia su valor y su tipo al reasignarla. También muestra qué ocurre cuando se intenta redeclarar con let o reasignar usando const. La idea es que cada paso se lea casi como si fuera una explicación de clase.

Objetivo del ejercicio

El código se ejecuta con JavaScript puro y deja la salida tanto en pantalla como en la consola del navegador para que puedas seguir la evolución de myVar paso a paso. La misma variable va cambiando para que se vea con claridad que en JavaScript una variable puede almacenar distintos tipos de dato.

Paso 1

Crear la variable

Se declara let myVar = 0 como punto de partida del ejercicio, porque let sí permite reasignar el valor más adelante.

Pasos 2-5

Reasignar e inspeccionar

La misma variable pasa por 0, 4.5, "John Doe" y false, comprobando siempre su contenido y typeof para ver cómo cambia el tipo.

Paso 6

Comparar let y const

Se documenta qué error aparece al intentar redeclarar con let o al intentar cambiar el valor de una constante.
