DOM + eventos avanzados

Delegacion de eventos con una zona de elementos clicables.

Esta practica usa un unico listener sobre un contenedor para capturar eventos de varios elementos hijos. Cada caja tiene un data-item-id y al interactuar se muestra que elemento disparo el evento, cuantas veces se ha pulsado y cual fue el ultimo tipo de interaccion.

Que se practica

Se combinan event.target, closest(), atributos data-* y estado visual para trabajar eventos avanzados sin repetir listeners por cada bloque.

Delegacion

Un listener para todo

El manejador vive en el contenedor y detecta sobre que caja se hizo click o accion de teclado.

Dataset

Identificar cada item

Cada caja define su identificador con data-item-id="a|b|c|d".

Estado

Feedback inmediato

La interfaz marca la caja activa, acumula contador por item y deja traza en una salida tipo consola.

Zona interactiva

Haz click en una caja o navega con Tab y pulsa Enter o Espacio para disparar eventos.
