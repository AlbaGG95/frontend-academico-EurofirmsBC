Arrays + DOM basico

Mi lista de peliculas con renderizado dinamico.

Este ejercicio practica la parte mas basica de arrays y manipulacion del DOM: recorrer una lista de titulos, crear nodos <li> y mostrarlos en pantalla dentro de un <ul> junto al contador total de peliculas.

Pasos del ejercicio

La practica esta pensada para hacer el recorrido completo desde el array hasta el DOM usando un bucle for.

Paso 1

Crear el array

Declarar peliculas con al menos cinco titulos.

Paso 2

Recorrer la coleccion

Usar un bucle for para leer cada titulo.

Paso 3

Crear elementos

Generar cada <li> con document.createElement.

Paso 4

Asignar contenido

Cargar el titulo con .textContent en cada elemento.

Paso 5

Anadir al UL

Insertar cada item en #lista-peliculas con .appendChild.

Paso 6

Actualizar contador

Mostrar en #contador cuantas peliculas hay en total.
