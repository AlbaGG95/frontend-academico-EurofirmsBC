HTML + eventos + validacion

Validacion de formularios con un numero entre 10 y 20.

Esta practica pide crear un formulario en HTML y capturar su envio con JavaScript para comprobar dos cosas: que el valor sea numerico y que este dentro del rango entre 10 y 20. Si la validacion falla se informa por consola; si pasa, se muestra el numero correcto introducido.

Que se practica

El ejercicio mezcla estructura HTML, escucha del evento submit, validacion con isNaN y conversion de texto a numero con parseInt.

Formulario

Capturar el envio

Se usa un addEventListener("submit") para interceptar el envio del formulario y validar el dato antes de procesarlo.

Validacion

Comprobar el valor

Primero se revisa si el valor es numerico; despues se confirma que el numero este dentro del rango pedido en el enunciado.

Salida

Mensajes de consola

El script deja un mensaje de error o de exito en consola y tambien lo refleja visualmente dentro de la pagina.

Formulario interactivo

Introduce un numero, pulsa el boton y comprueba el comportamiento del validador. Prueba casos como abc, 8, 15 o 21.

Enviar numero

El campo acepta texto a proposito para que la validacion tenga que comprobar si lo recibido es realmente un numero.

Rango valido: desde 10 hasta 20, ambos incluidos.

Pendiente

Resultado visual
