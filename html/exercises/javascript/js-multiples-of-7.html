Bucle for + condicional if

Contar cuantos numeros del 1 al 100 son multiplos de 7.

Esta practica recorre los numeros del 1 al 100 con un for, usa un if para detectar si un numero es multiplo de 7 y va sumando un contador cada vez que la condicion se cumple. El resultado final se muestra en consola y tambien queda documentado visualmente dentro de la pagina.

Objetivo del ejercicio

La idea es practicar tres piezas basicas de JavaScript al mismo tiempo: recorrido con bucle, comprobacion condicional y acumulacion de resultados con una variable contador.

Paso 1

Recorrer del 1 al 100

El programa usa un for para avanzar numero a numero desde 1 hasta 100 sin saltarse ningun valor.

Paso 2

Comprobar cada numero

Dentro del bucle se aplica number % 7 === 0 para saber si el valor actual es multiplo de 7.

Paso 3

Contar coincidencias

Cada vez que la condicion se cumple, el contador aumenta en una unidad y el numero se registra para mostrarlo despues.
