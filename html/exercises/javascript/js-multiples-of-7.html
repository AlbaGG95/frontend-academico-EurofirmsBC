Bucle for + condicional if

Contar cuántos números del 1 al 100 son múltiplos de 7.

Esta práctica recorre los números del 1 al 100 con un for, usa un if para detectar si un número es múltiplo de 7 y va sumando un contador cada vez que la condición se cumple. El resultado final se muestra en consola y también queda documentado visualmente dentro de la página.

Objetivo del ejercicio

La idea es practicar tres piezas básicas de JavaScript al mismo tiempo: recorrido con bucle, comprobación condicional y acumulación de resultados con una variable contador.

Paso 1

Recorrer del 1 al 100

El programa usa un for para avanzar número a número desde 1 hasta 100 sin saltarse ningún valor.

Paso 2

Comprobar cada número

Dentro del bucle se aplica number % 7 === 0 para saber si el valor actual es múltiplo de 7.

Paso 3

Contar coincidencias

Cada vez que la condición se cumple, el contador aumenta en una unidad y el número se registra para mostrarlo después.
