Ejercicio HTML + script

Mensaje personal construido con variables dentro de un HTML.

Esta practica construye un mensaje personal a partir de una constante para el nombre, una variable numerica para la edad y una booleana para indicar si gusta programar. La logica esta separada en un archivo JS para mantener el proyecto mas limpio y escalable.

Que se pide en este ejercicio

El resultado se muestra en consola y tambien en una vista visual dentro de la pagina. La estructura deja el HTML para el contenido y el archivo JS para la logica del ejercicio.

Constante

Nombre

Se declara una constante con el nombre para que ese dato quede fijo desde el principio.

Variable

Edad numerica

La edad se guarda en una variable de tipo numero para usarla dentro del mensaje final.

Booleano

Gusto por programar

El valor true o false se aprovecha para construir una frase mas natural.
