sumarHasta(numero): validar y acumular desde 1.

Esta practica crea una funcion que recibe un numero, comprueba si es mayor que 0 y devuelve la suma acumulada entre 1 y ese valor. Si el dato no es valido, muestra un error y devuelve -1.

Que se practica

El ejercicio combina validacion basica, recorrido con for y acumulacion en una variable para resolver una operacion sencilla pero muy tipica en JavaScript.

Paso 1

Validar el parametro

Se comprueba que el valor recibido sea un numero finito y mayor que 0 antes de hacer cualquier suma.

Paso 2

Recorrer del 1 al numero

Un bucle for avanza numero a numero para ir agregando cada valor a la suma total.

Paso 3

Devolver el resultado

Si el dato es valido, la funcion devuelve la suma; si no lo es, deja un error en consola y devuelve -1.
