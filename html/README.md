# HTML

Esta carpeta contiene las paginas reales del proyecto.

## Subcarpetas

- `hub/`: entradas principales del sitio
- `docs/`: documentacion tecnica y teoria
- `exercises/html-css/`: practicas del bloque de HTML y CSS
- `exercises/javascript/`: practicas del bloque de JavaScript

## Regla de uso

- si una pagina es parte de la navegacion principal, va en `hub/`
- si una pagina explica teoria, guia o metodologia, va en `docs/`
- si una pagina es una practica o ejercicio, va en `exercises/`

## Nota

Las paginas de la raiz del proyecto no contienen el contenido real. Solo
redirigen a estas rutas para mantener una entrada comoda al abrir el proyecto.
